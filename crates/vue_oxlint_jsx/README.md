# vue_oxlint_jsx

A transformer that transforms AST returned by [`vue_oxlint_jsx`](https://github.com/liangmiQwQ/vue-oxlint-toolkit/tree/main/crates/vue_oxlint_jsx) into a JS/TSX representation.
