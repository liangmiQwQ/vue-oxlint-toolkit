# Vue Oxlint Toolkit

An experimental toolkit to provide high performance Vue linting support.

> [!IMPORTANT]
> This project is an early proof of concept. It can't be used directly for now.

## Credits

Includes a fork of [vue-oxc-parser](https://github.com/zhiyuanzmj/vue-oxc-parser) by zhiyuanzmj.

Includes a rust port of [vue-eslint-parser](https://github.com/vuejs/vue-eslint-parser).
