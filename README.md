# Vue Oxlint Toolkit

An experimental toolkit to provide high performance Vue linting support.

> [!IMPORTANT]
> This project is an early proof of concept. It can't be used directly for now.

## Credits

This project includes a forks of [vue-oxc-parser](https://github.com/zhiyuanzmj/vue-oxc-parser) (by zhiyuanzmj) and [oxc_codegen](https://github.com/oxc-project/oxc) (by VoidZero Inc.).
